# 2D & 3D Geometric Transformations Lab 

## Project Overview

The implementation uses **Python, OpenCV, and NumPy** to visualize and experiment with:

- Basic 2D transformations
- Reflection transformations
- Shearing transformations
- Advanced 2D transformation matrices
- 3D homogeneous coordinate matrix operations


**2D နှင့် 3D Geometric Transformations** များကို Python, OpenCV နှင့် NumPy အသုံးပြုပြီး လက်တွေ့စမ်းသပ်ထားပါတယ်။

ပါဝင်သော Transformations များမှာ:

- Basic Transformations
- Reflection
- Shearing
- 2D Advanced Transformation Matrices
- 3D Homogeneous Matrix Multiplication

# ⚙️ How to Setup and Run

## 1. Install Dependencies

Before running the project, install the required Python libraries.

Open **Command Prompt (CMD)** or **PowerShell** and run:

pip install opencv-python numpy matplotlib

2. Clone the Repository

Clone this repository into your local machine:
git clone <your-repository-link-here>

3. Run the Program

Run the Python files using:
2D Transformations-
python geo2d.py
3D Transformations-
python geo.py

*********
Input Image (Optional)

To test transformations with your own image:
Add an image into the project root directory.
Rename it as - test_image.jpg
